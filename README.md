# String_Sum_Concatenate

ADD Function Which Take 2 Arguments. If Both The Arguments Are Integer Then Their sum returned. If Both Are Strings they will be
Concatenated And Resultant String returned. ;)

Eg1.
  Input 1 is:45
  Input 2 is:55
  Sum is:100

Eg2.
  Input 1 is:K5
  Input 2 is:55
  Concatenate is:K555
